Mawari マワリ

あと◯分使えるなら、ここを歩こう。回り道で歩いて、交通費を節約。

      ⚙️ 運賃・通過駅の設定（ODPTキー）

      入力すると、運賃データの無い区間（東武・京成・私鉄等）の実額運賃と、 各乗車区間の通過駅を ODPT から補完します。キーはこの端末のブラウザ（localStorage）にのみ保存され、外部送信・リポジトリ保存はしません。

      回り道の候補

      最短経路と比べた「追加時間」と「運賃差額」を表示します（節約＝マイナス）。

      このアプリについて / 既知の制約
      • 経路データは無料の Transit API (ls8h) をブラウザから直接利用しています（無認証・読み取り専用）。
      • 運賃はフィード依存です。東京メトロ・都営・路線バス等は運賃が出ますが、JR各線・新幹線・私鉄の一部は運賃データが無く「運賃不明」になります。経路に運賃不明の区間が1つでも含まれると、その経路全体の運賃が出ません。ODPTキーを設定すると、東武など私鉄/地下鉄の実額運賃と各乗車区間の通過駅を補完します（JR等は ODPT でも欠ける場合あり）。
      • 色分け：各乗車区間は路線色で表示します（API の路線色を使用）。
      • 通過駅スライダー（ODPTキー設定時）：経路が通過する駅を選ぶと、そこで降りて目的地まで歩く経路に再検索し、運賃と割引額を更新します。
      • 歩行距離・歩数・カロリーは推定値です（徒歩時間 × 約80m/分 で換算）。
      • 回り道候補は、複数経路の列挙と「鉄道を避ける」検索を組み合わせて生成しています（PoC段階）。
